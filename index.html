Independent Chat Infrastructure

Gaia Launcher and Current Server are built for small communities that want a clear desktop entry point and a server they can inspect, host, and move.

Own the place where your group talks.

Gaia Launcher keeps the everyday flow simple: choose a server, sign in, join channels, use voice, and stay current with launcher and server releases.

Current Server handles roles, channels, moderation, invites, messages, and voice-room state. The project is open source so server owners can review the code and choose the hosting setup that fits their group.

What runs on your machine

Gaia Launcher is the desktop shell. Current Server is the hostable service that stores community configuration and message surfaces for the servers you run.

How people connect

A server owner publishes a Current Server, then shares its address with members. Gaia keeps the server list, account handoff, and update path in one place.

Why the public site is separate

This website is only for public product information, releases, policy pages, and source links. Private chat content stays inside the launcher and the server.
